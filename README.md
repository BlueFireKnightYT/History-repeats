# History repeats
This is a little prototype of mine.
 I am making this completely on my own.
 
 The idea is that you have 10 seconds to do what you want. Lets call that cylce 1.
 when the timer is over, You will return to the current checkpoint and you will be in cycle 2.
 You have a ghost in cycle 2 who will repeat what you did in cycle 1. with that, you will need to solve puzzles with your ghost.
